Supported Platforms
===================

The C++ SDK is compatible with the following operating systems:

* Ubuntu 20.04+ 
	* x86_64
	* gcc9/gnustl, clang10/libc++
* Debian 11 (Beta)
	* arm64
	* gcc9/gnustl
* MacOS 10.15+ 
	* x86_64, arm64
* Windows 10+
	* x86_64
	* Windows SDK 10.0.20348.0+
